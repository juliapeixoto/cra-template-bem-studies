# BEM Studies

Template for BEM CSS studies.

## Installation

Open the terminal, in any folder that you want to save the project and run the command:

```bash
  npx create-react-app awesome-app --template bem-studies
```

## Run Locally

Go to the project directory

```bash
  cd awesome-app
```

Start the server

```bash
  yarn start
```
